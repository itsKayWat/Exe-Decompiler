Exe-Decompiler

A powerful GUI tool for decompiling and analyzing Electron-based executable applications.

Purpose

Exe-Decompiler was created to assist developers and security researchers in analyzing, modifying, and understanding Electron-based applications. Through its user-friendly interface, users can extract, analyze, and recompile Electron applications with ease.

Key Features

  • Extract ASAR archives from Electron applications
  • Analyze source maps for better code understanding
  • Modify application files with built-in tools
  • Recompile changes back into the application
  • Development tools integration
  • Modern dark-themed GUI interface

System Requirements

  • Windows Operating System
  • Python 3.7 or higher
  • Node.js (Latest LTS version)
  • Administrator privileges (for certain operations)

Installation Guide

  1. Download and install Python 3.7+ from python.org
  2. Download and install Node.js from nodejs.org
  3. Clone this repository to your local machine
  4. Run requirements.py to install all dependencies
  5. Launch exe_decompiler.py to start the application

Usage Example

Scenario: Analyzing a desktop application built with Electron to understand its functionality or modify its behavior.

Steps:

  1. Launch Exe-Decompiler
  2. Browse to select the application folder
  3. Click "Extract ASAR" to decompile
  4. Use "Edit Files" to modify the code
  5. Click "Recompile Changes" to rebuild

Common Use Cases

  • Application analysis for security research
  • Debugging Electron applications
  • Learning how applications are built
  • Modifying application behavior
  • Adding custom features to existing applications
Important Note: Always ensure you have the legal right to modify applications before using this tool. This tool is intended for legitimate research and development purposes only.
